**Суть:** Упражнение на работу с параллельными массивами и условные присваивания.

## 📁 **functions.cpp** — Табулирование функции с заданной точностью
**Что делает:**
- Вычисляет функцию `f(x)` разложением в ряд:
  ```
  f(x) = sum(k=0 to ∞) [(-1)^k / ((k+1)!)^2] * (x/2)^(2(k+1))
  ```
- Использует **рекуррентный метод** (каждый следующий член через предыдущий)
- Строит таблицу значений на интервале `[a, b]` с шагом `h`
- Сохраняет результаты в файл `function_table.txt`

**Суть:** Численные методы, ряды, работа с файлами. **Самая серьезная работа в подборке!**
